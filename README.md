# Crypto Market Dashboard

Realtime crypto dashboard (Binance public API) — React + TypeScript UI with MobX state, shadcn/ui + Tailwind, and lightweight-charts.

## Quick start

1. Install deps

```bash
npm install
```

2. Dev server (HMR)

```bash
npm run dev
```

3. Build / preview

```bash
npm run build
npm run preview
```

Node recommendation: Node 20.19+ or 22.12+ (Vite 8).
